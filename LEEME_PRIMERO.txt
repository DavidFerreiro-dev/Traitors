┌──────────────────────────────────────────────────────────────────────┐
│  🎯 CÓMO JUGAR                                                       │
└──────────────────────────────────────────────────────────────────────┘

  HOST (Creador):
    1. Crear sala → Obtener código
    2. Compartir código con amigos
    3. Iniciar partida (mínimo 3 jugadores)
    4. Iniciar votación cuando estén listos
    5. Ver resultados y reiniciar

  JUGADORES:
    1. Unirse con código
    2. Recibir rol (Impostor o Inocente)
    3. Discutir en el chat
    4. Votar al sospechoso
    5. Ver resultados


┌──────────────────────────────────────────────────────────────────────┐
│  🌐 JUGAR EN RED LOCAL                                               │
└──────────────────────────────────────────────────────────────────────┘

  Windows:
    1. Abrir CMD o PowerShell
    2. Ejecutar: ipconfig
    3. Buscar "Dirección IPv4" (ej: 192.168.1.100)
    4. Compartir: http://192.168.1.100:3000

  Mac/Linux:
    1. Abrir Terminal
    2. Ejecutar: ifconfig
    3. Buscar IP local
    4. Compartir: http://TU_IP:3000


┌──────────────────────────────────────────────────────────────────────┐
│  🔧 SOLUCIÓN DE PROBLEMAS                                            │
└──────────────────────────────────────────────────────────────────────┘

  ❌ "npm no se reconoce"
     → Instalar Node.js desde: https://nodejs.org/

  ❌ "Puerto 3000 ya en uso"
     → Cerrar otros servidores o cambiar puerto en server.js

  ❌ "Cannot find module"
     → Ejecutar: npm install

  ❌ Jugadores no pueden conectarse
     → Verificar que estén en la misma red WiFi
     → Usar la IP correcta del servidor
